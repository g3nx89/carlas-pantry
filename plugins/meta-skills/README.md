# Meta-Skills Plugin

A collection of meta-skills for mastering Claude Code, MCP servers, and agentic workflows.

## Overview

This plugin provides skills and utilities to help users get the most out of Claude Code and its ecosystem:

- **Claude Code mastery** - Tips, patterns, and best practices
- **MCP server utilization** - Configuration, troubleshooting, and optimization
- **Agentic workflows** - Patterns for effective multi-agent coordination
- **Productivity techniques** - Workflows and shortcuts for common tasks

## Installation

```bash
claude plugins add carlas-pantry/meta-skills
claude plugins enable meta-skills
```
